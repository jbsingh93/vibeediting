# Brand voice — how the agent writes for you

The agent reads this file (plus `brand/brand.json` → `tone`) before writing ANY
on-screen copy, voice-over script, caption emphasis, CTA or thumbnail headline.
Replace the guidance below with your own voice — or ask the agent to interview
you and rewrite this file.

## The dials (set in brand.json, explained here)

- **`tone.register`** — `casual` (contractions, direct address, humor OK) ·
  `professional` (clear, confident, no slang) · `playful` (energy, wordplay,
  emoji allowed in captions).
- **`tone.sellStyle`** — the soft-sell ↔ hard-sell spectrum:
  - `soft` — never command the viewer to buy. Invite, demonstrate, let proof
    speak. No "BUY NOW", no countdown pressure, no ALL-CAPS urgency.
  - `neutral` — clear CTAs ("Try it free", "Learn more") without pressure
    tactics. The default.
  - `direct` — conversion-first: strong verbs, urgency and scarcity are
    allowed, still honest (no fake countdowns, no fabricated claims).
- **`tone.language`** — write all viewer-facing copy in this language.
  Keep technical/file naming in English regardless.

## House rules (keep these — they're craft, not branding)

1. **Hooks earn the first 3 seconds.** Lead with the viewer's problem or a
   concrete payoff — never with the brand name.
2. **One idea per caption line.** 4–7 words per line for 9:16 reels; the
   emphasis word is the one you'd stress out loud.
3. **Claims need receipts.** Numbers, names and "studies show" must come from
   the brief or verified research — never invented (anti-fabrication rule).
4. **CTAs match the platform.** YouTube → Subscribe; LinkedIn → Follow;
   landing-page traffic → the page's actual verb. Never a generic "comment below".
5. **Read it aloud.** If a sentence trips the tongue, it trips the voice-over.
