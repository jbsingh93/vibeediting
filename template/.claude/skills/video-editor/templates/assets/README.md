# Assets

This folder holds your own brand and proof assets that the briefs and compositions
reference — things like testimonial JSON files.

- `testimonials/` — customer-quote JSON files for quote cards and ad proof scenes.
  See `testimonials/README.md` for the format. A fictional example ships there.

Brand identity (colors, fonts, voice ID) lives in `brand/brand.json`, not here.
Logos live alongside the brand config — see `brand/brand.json` `logoPath`.

Everything you add here is your own content and is never shipped with the package.
